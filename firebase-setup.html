To enable ROM syncing between your phone and watch, you need to configure Firebase Realtime Database rules.

Step 1: Go to Firebase Console

Visit https://console.firebase.google.com

Select your project: weargb-input

Step 2: Navigate to Realtime Database

In the left sidebar, click on Realtime Database

Click on the Rules tab

Step 3: Update Security Rules

Replace the existing rules with:

{
  "rules": {
    "url_sessions": {
      "$sessionId": {
        ".read": true,
        ".write": true,
        ".indexOn": ["timestamp"]
      }
    }
  }
}

Step 4: Publish Rules

Click the Publish button to save your changes

‚ö†Ô∏è Important Note

These rules allow anyone with a session ID to read/write data. This is fine for development and personal use. For production apps with many users, you should implement proper authentication.

Alternative: Test Mode (Temporary)

For quick testing (30 days), you can use these open rules:

{
  "rules": {
    ".read": true,
    ".write": true
  }
}

Note: Even without Firebase configured, you can still:

  • Select ROMs from the dropdown list
  • Use the Pro version's custom URL feature
  • The app will show the URL for manual entry if needed
